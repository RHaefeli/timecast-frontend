What is this application about?

This application is made in order of a school project.

The application is made to manage employees with their corresponding projects. It is possible to create new Employees and new Projects. An Employee needs a Contract to be able to be assigned to a project. With creating Allocations an employee can then be assigned to a project according to the employees pensum.
